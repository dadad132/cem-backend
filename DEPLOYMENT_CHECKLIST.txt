STEP 1: UPLOAD FILES TO UBUNTU
-------------------------------
Option A - Using SCP:
  scp -r crm-backend/ user@server-ip:/home/user/

Option B - Using Git:
  ssh user@server-ip
  git clone <your-repo> crm-backend

Option C - Using SFTP/FTP:
  Upload entire folder to /home/user/crm-backend


STEP 2: CONNECT TO SERVER
--------------------------
  ssh user@your-server-ip
  cd crm-backend


STEP 3: RUN DEPLOYMENT
-----------------------
  chmod +x deploy.sh
  ./deploy.sh

  (This will automatically:
   - Install Python 3.12
   - Create virtual environment
   - Install all packages from requirements.txt
   - Setup directories
   - Configure permissions)


STEP 4: START SERVER
--------------------
  ./start_ubuntu.sh

  OR for background:
  source venv/bin/activate
  nohup python start_server.py > logs/server.log 2>&1 &


STEP 5: CONFIGURE FIREWALL
---------------------------
  sudo ufw allow 8000/tcp
  sudo ufw reload


STEP 6: ACCESS YOUR CRM
------------------------
  http://YOUR-SERVER-IP:8000
